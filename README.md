# TGM-Renitify-Rental-Mobile-Application

##  Overview

TGM Rentify is a mobile rental management application developed as part of an academic software engineering project. The platform connects tenants and landlords through a centralized system for property listing, property discovery, and rental management.

As a member of the development team, my primary contribution focused on the **Landlord Module**, including property management, image handling, and property search functionality.

---

##  Project Objectives

* Simplify the rental property management process
* Provide a user-friendly platform for landlords and tenants
* Enable efficient property listing and searching
* Improve accessibility to rental information
* Utilize modern mobile development technologies

---

##  My Contribution

### Landlord Property Management

Implemented complete property management functionality for landlords, including:

* Add new property listings
* Edit existing property details
* Delete property listings
* View landlord-owned properties
* Manage property information efficiently

###  Property Image Management

Developed image upload functionality using Firebase Storage:

* Upload property images from mobile devices
* Display uploaded images within property listings
* Store and retrieve images securely

###  Property Search & Filtering

Implemented property search features allowing users to:

* Search properties by keywords
* Filter properties by location
* Filter properties by property type
* View matching search results dynamically

###  Data Isolation

Ensured landlords can only access and manage their own properties through Firebase-based user filtering.
